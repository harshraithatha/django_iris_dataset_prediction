# This is Django project created on the Iris dataset

It can take below inputs and predict the Iris flower
- Sepal length
- Sepal width
- Petal length
- Petal width

### Accuracy measured with 'random_state=1' = 96.7%
### Accuracy measured with 'random_state=9' = 98.9%
